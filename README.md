## Project Scope
A Hybrid Mobile App (Native Shell + WebView) designed for the Team Builder + Skill Match scenario.

### Key Features
* **Hybrid Architecture:** React web layer wrapped in Capacitor for native capabilities.
* **Skill Matching Database:** Robust tracking of user Profiles (skills/availability), Projects, and proposed Matches.
* **Core Mobile Capabilities:** Biometric Login placeholder, Light/Dark thematic support, and Offline Cache logic.
* **Interactive UI:** Voice Input functionality in the Chat Interface.
* **Agentic AI Workflow:** An AI Concierge that interprets goals (e.g., 'Build my fintech team'), actively finds cross-functional matches, drafts introductory messages, explains its reasoning visually, and strictly mandates user approval before dispatching actions.
